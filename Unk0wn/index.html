What is Unk0wn?

Unk0wn is a Discord Bot Moderation Tool for small to moderate-sized Discord servers, built using Discord.py. It offers features such as:

  • Auto-Moderation
  • Moderation Tools
  • Ticketing
  • Levelling

Available Slash Commands

  • /setup-help – Step-by-step guide for setting up the bot
  • /logging – Enable or disable logging
  • /createlogs – Create category and channels for logs
  • /automod-settings – Configure auto-moderation
  • /kick – Kick a member with a reason
  • /ban – Ban a member with a reason
