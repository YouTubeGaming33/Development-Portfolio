What is Unk0wn?

Unk0wn is a Discord Bot Moderation Tool for small to moderate-sized Discord servers, built using Discord.py. It offers features such as:

  • Auto-Moderation
  • Moderation Tools
  • Ticketing
  • Levelling

Available Slash Commands

  • /setup-help - Step-by-step Guide for Correctly Setting Up the Bot.
  • /logging - Disable or Enable Logging.
  • /createlogs - Creates Category and Channels for Logging.
  • /automod-settings - Change Auto-Moderation Settings,
  • /kick - Used to Kick a Member with a Reason.
  • /ban - Used for Banning a Member with a Reason.
