What is George?

George is a Discord Bot for the AMT Community Discord Server, run by OC01. It offers features such as:

  • Auto-Moderation
  • Moderation Tools
  • Ticketing
  • Levelling
  • Missions
  • Auto-responders

Available Slash Commands

Find all Relevant Unk0wn Commands for Moderation, in additon to Other Commands.

  • /setup-help – Step-by-step guide for setting up the bot
  • /logging – Enable or disable logging
  • /createlogs – Create category and channels for logs
  • /automod-settings – Configure auto-moderation
  • /kick – Kick a member with a reason
  • /ban – Ban a member with a reason
