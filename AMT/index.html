What is George?

George is a Discord Bot for the AMT Community Discord Server, run by OC01. It offers features such as:

  • Auto-Moderation
  • Moderation Tools
  • Ticketing
  • Levelling
  • Missions
  • Auto-responders

Available Slash Commands for A Mythical Tale

Find Below all the Slash Commands which can be ran inside of the Mythical Tale Discord

  • /claim – Claim a Bounty via Number Indentifer
  • /set-bounty - Set a Bounty on a User
  • /close-bounty – Close a Bounty upon Completetion
  • /create-listing – Create a Listing to sell an Item in-game
  • /daily – Receive a Daily Mission
  • /weekly – Receive a Weekly Mission
  • /profile – View yours or someone elses Profile
  • /help - Provides information on receiving help both in AMT and ATT
  • /open-ticket - Open a ticket with the Moderation Team
